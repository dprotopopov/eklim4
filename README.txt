1. Скачать ImageMagick со страницы http://www.imagemagick.org/script/binary-releases.php
2. Установить скачанный архив в папку
3. Прописать путь к папке ImageMagick в файле imagemagick.bat
4. Поместить конвертируемые файлы в папку input
5. Выполнить imagemagick.bat
6. Забрать сконвертированные файлы из папки output
